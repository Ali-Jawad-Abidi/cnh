## About
TheCNH.co.uk is a web application that showcases the products and services offered by TheCNH company. It provides users with a modern and responsive interface to explore the company's offerings, place orders, and manage their accounts.

## Features
- Browse and search products
- View detailed product information
- Add products to the shopping cart
- Manage the shopping cart (add, remove, update quantities)
- Proceed to checkout and place orders
- User authentication and account management
- Order history and order tracking
- Admin dashboard to manage products, orders, and users

## Technologies
The project is built using the following technologies:

- **MongoDB**: A popular NoSQL database used for storing application data.
- **Express.js**: A fast and minimalist web application framework for Node.js.
- **React.js**: A JavaScript library for building user interfaces.
- **Node.js**: A JavaScript runtime environment used for server-side development.
- **Tailwind CSS**: A highly customizable CSS framework for building modern UIs.
- **Other React Libraries**: Various additional libraries and dependencies are used for state management, routing, form handling, authentication, etc.

## Getting Started
To get a local copy of this project up and running, follow the steps below.

## Installation
1. Clone this repository to your local machine using:
   ```
   git clone https://github.com/your-username/TheCNH.co.uk.git
   ```
2. Navigate into the project's root directory:
   ```
   cd TheCNH.co.uk
   ```
3. Install the dependencies using a package manager such as npm or yarn:
   ```
   npm install
   ```
   or
   ```
   yarn install
   ```

## Usage
1. Start the development server:
   ```
   npm start
   ```
   or
   ```
   yarn start
   ```
2. Open your web browser and visit `http://localhost:3000` to see the application running locally.
